About application
Student Performance Analysis and Visualization

Student Performance Analysis System provides a interface for college maintenance. It can used by educational institutes to analyse the student performance easily . The creation, analysation, up to date information regarding academic career of a student is critically very important in an institution. This System deals with all kinds of student related academic details. It keeps track on all details of a student from the day one to the end of the academics . This project is mainly focuses on Student performance. In this system, Firstly data is collected by the students from their previous academic performance and the data is stored in this system. Based on the previous data, student can predict their next academic performance. This performance analysis is shown in the form of graphs using pie-charts and bar-graphs.
